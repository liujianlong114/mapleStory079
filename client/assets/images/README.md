## 七、图片规范

- **格式**：PNG-32（带 Alpha 通道）
- **色深**：24-bit RGB + 8-bit Alpha
- **推荐工具**：
  - `WzRepacker` / `HaRepacker`（从 wz 解包）
  - `TexturePacker`（生成精灵图集）
  - `Aseprite`（像素风修改）
- **总大小预估**：约 500–800 MB（原始资源），可压缩至 200 MB 左右
- **加载配置**：已在 `pubspec.yaml` 中声明 `assets/images/`

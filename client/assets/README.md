```

```

## 🎯 制作优先级建议

### P0（立即需要）

- 登录界面贴图（登录/角色选择/加载画面）
- 6 职业 × 基础动作精灵（idle/walk/attack）
- 基础怪物（蜗牛/蓝蜗牛/绿蘑菇）
- 基础 UI（状态条/按钮/对话框）
- 基础战斗音效（攻击/受击/升级）
- 基础 UI 音效（点击/弹窗）
- 金币/药水/装备图标

### P1（近期需要）

- 完整 6 职业全动作动画
- 常见村庄 BGM（射手村/魔法密林）
- 技能图标（各职业核心技能）
- 物品装备图标（各职业武器/防具）
- 场景背景（彩虹岛/明珠港）

### P2（后期完善）

- 高级地图/副本
- Boss 战/Boss 精灵
- NPC 头像和特效
- 宠物/坐骑精灵

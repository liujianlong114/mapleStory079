## 四、资源规范

| 项目 | 规范 |
|------|------|
| BGM 格式 | MP3，44.1 kHz，立体声，128 kbps |
| SFX 格式 | OGG（Vorbis），44.1 kHz，单声道/立体声，96 kbps |
| 命名规则 | `bgm_<地图名>.mp3` / `sfx_<事件>.ogg` |
| 循环 | BGM 需可无缝循环，SFX 一次性播放 |
| 预估总量 | BGM 约 25 首 ≈ 70 MB；SFX 约 50 个 ≈ 5 MB |

## 五、获取方式

1. 官方 wz 资源提取：使用 `HaRepacker` / `WzRepacker` 从 `Sound.wz` 提取 `Bgm00.img` ~ `Bgm10.img` 与 `Sound*.img`
2. 开源替代：[freesound.org](https://freesound.org) / [opengameart.org](https://opengameart.org) / Kevin MacLeod (incompetech.com) 的 CC-BY 音乐
3. 风格参考：网易云音乐 / Bilibili 冒险岛 079 BGM 合集（用于确认曲目风格）

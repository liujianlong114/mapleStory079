## 五、资源规范

- **音频格式**：OGG (Vorbis) / MP3
- **采样率**：44.1 kHz
- **比特率**：96–192 kbps（背景音乐建议 128 kbps，SFX 建议 96 kbps）
- **声道**：立体声
- **文件命名**：`[类型]_[地图ID|名称].[ogg|mp3]`
- **总大小预估**：约 200–300 MB

## 六、获取方式

1. **从官方客户端 wz 资源提取**
   - 使用 `WzRepacker`、`HaRepacker` 等工具
   - 提取 `Sound.wz` 中的 `Bgm00.img`、`Bgm01.img` 等
2. **使用 CC0 / 开源资源替代**
   - [freesound.org](https://freesound.org) 搜索 fantasy、rpg、town
   - [opengameart.org](https://opengameart.org) 提供大量免费游戏音效
   - [incompetech.com](https://incompetech.com) Kevin MacLeod 的 CC-BY 音乐
3. **音乐合成 / 重新编曲**（长期规划）
   - 使用 LMMS / FL Studio 对经典曲目进行重制

## 七、加载配置

在 `pubspec.yaml` 中已声明：
```yaml
assets:
  - assets/audio/
```

Flutter 端通过 `flame_audio` / `audioplayers` 加载：
```dart
AudioPlayer().play(AssetSource('audio/00002000.ogg'));
```

资源加载器示例：

```dart
import 'package:audioplayers/audioplayers.dart';

class AudioAssets {
  static final Map<String, String> all = {
    'bgm_maple_island': bgmMapleIsland,
    'bgm_lith_harbor': bgmLithHarbor,
    // ... 其他音频
    'sfx_revive': sfxRevive,
  };

  static Future<void> play(String key) async {
    if (all.containsKey(key)) {
      await AudioPlayer().play(AssetSource(all[key]!.replaceFirst('assets/', '')));
    }
  }
}
```

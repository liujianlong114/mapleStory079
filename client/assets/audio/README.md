> 资源路径：`client/assets/audio/`
> BGM 作曲：StudioEIM（韩国官方曲目库）
> 原始数据整理自：MapleStory v0.79 wz Sound 资源库 / HeavenMS handbook Map.txt 地图索引

## 五、资源规范

- **音频格式**：OGG (Vorbis) / MP3
- **采样率**：44.1 kHz
- **比特率**：96–192 kbps（背景音乐建议 128 kbps，SFX 建议 96 kbps）
- **声道**：立体声
- **文件命名**：`[类型]_[地图ID|名称].[ogg|mp3]`
- **总大小预估**：约 200–300 MB

## 六、获取方式

1. **从官方客户端 wz 资源提取**
   - 使用 `WzRepacker`、`HaRepacker` 等工具
   - 提取 `Sound.wz` 中的 `Bgm00.img`、`Bgm01.img` 等
2. **使用 CC0 / 开源资源替代**
   - [freesound.org](https://freesound.org) 搜索 fantasy、rpg、town
   - [opengameart.org](https://opengameart.org) 提供大量免费游戏音效
   - [incompetech.com](https://incompetech.com) Kevin MacLeod 的 CC-BY 音乐
3. **在线音乐合集参考**
   - [网易云音乐-冒险岛音乐全集](https://music.163.com/playlist?id=53144841) 含明珠港/魔法密林/射⻰等全部 BGM
   - [哔哩哔哩 冒险岛日本 BGM 合集](https://www.bilibili.com/video/BV14d4y1Y7hu/) 可作为风格参考
4. **音乐合成 / 重新编曲**（长期规划）
   - 使用 LMMS / FL Studio 对经典曲目进行重制

## 七、加载配置

在 `pubspec.yaml` 中已声明：
```yaml
assets:
  - assets/audio/
```

Flutter 端通过 `flame_audio` / `audioplayers` 加载：
```dart
AudioPlayer().play(AssetSource('audio/Bgm00/MapleRoad.ogg'));
```

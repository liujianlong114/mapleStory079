## 🎯 高优先级（必须完成）

### 任务 1: 补充缺失的后端路由接口（问题 #1）
- **说明**: `api_service.dart` 调用的 6 个接口在 `routes.go` 中不存在。需在后端补充并连接到对应 service。
- **具体内容**:
  - `POST /api/v1/combat/calculate-damage` → `combat_service.go` 新增 `CalculateDamage()` handler 封装
  - `POST /api/v1/combat/player-attack-mob` → 复用 `PlayerAttackMob()`
  - `POST /api/v1/combat/mob-attack-player` → 复用 `MobAttackPlayer()`
  - `GET /api/v1/combat/stats` → 读取 `CharacterStats` 或计算
  - `GET /api/v1/items` → 新增 item repository 读取物品列表
  - `GET /api/v1/items/:id` → 物品详情
  - `GET /api/v1/npcs/map/:mapId` → 按地图筛选 NPC
- **涉及文件**: `internal/handler/routes.go`、`internal/handler/game_handler.go`（或新建 `combat_handler.go`/`item_handler.go`）、`internal/repository/item_repository.go`
- **验收标准**: `api_service.dart` 所有 HTTP 请求在 `routes.go` 中都有对应 handler；`curl` 测试所有接口返回 200

### 任务 2: 统一 WebSocket 消息协议格式（问题 #2）
- **说明**: 服务端与客户端的消息 JSON 结构不一致。
- **具体内容**:
  - 方案 B（推荐）: 服务端接受 `{"type", "payload": {...}, "sender_id", "room"}` 格式，将原 `routeMessage` switch 的字段读取改为从 `msg.Payload` 读取
  - 在 `pkg/utils/constants.go` 新增 `WSMessageType*` 常量，统一字符串标识
  - 客户端和服务端都引用同一份常量（Dart 手动同步，建议在 `api_service.dart` 顶部加类型映射）
  - 添加心跳超时：客户端 30s ping，服务端 60s 未收到 pong 则断开
- **涉及文件**: `internal/handler/websocket_handler.go`、`client/lib/services/websocket_service.dart`、`pkg/utils/constants.go`
- **验收标准**: 手动测试 `connect → sendChat → receiveChat → sendPosition → disconnect` 全流程正常，无字段解析错误

### 任务 3: 补充缺失的战斗接口 handler
- **说明**: `game_handler.go` 只有 `Attack`，需补充 `GetCombatStats`、`ReviveCharacter`。
- **具体内容**:
  - `GET /api/v1/combat/stats?character_id=123` → 返回 `{maxHP, maxMP, physicalAttack, magicAttack, physicalDefense, magicDefense, hitRate, critRate}`
  - `POST /api/v1/combat/revive` → `character_id` 复活，HP = maxHP * 0.25, MP = maxMP * 0.25
- **涉及文件**: `internal/handler/game_handler.go`、`internal/handler/routes.go`
- **验收标准**: curl 成功调用，响应状态码 200，返回字段与 Dart `models/combat.dart` 一致

## 🎯 中优先级（建议完成）

### 任务 4: 放置 CC0 占位资源（音频/图片/精灵）
- **说明**: `client/assets/` 下只有 README 清单，资源文件需填充。
- **具体内容**:
  - `audio/`: 从 freesound.org / incompetech.com 下载 CC0 音频，重命名为 README 中列出的文件名，OGG 格式
  - `images/`: 用 Aseprite / GIMP 绘制 2-3 种像素风的 UI 组件（标题/按钮/HP条/面板）作为占位
  - `sprites/`: 用 64×64 像素的简单人物/怪物精灵作为占位，按职业/方向分目录
  - 在 `pubspec.yaml` 中声明 `assets/` 路径
  - 修改 `game_world.dart` 的 `SpriteLoader` 真正加载 assets 资源
- **涉及文件**: `client/assets/*`、`client/pubspec.yaml`、`client/lib/game/engine/sprite_loader.dart`
- **验收标准**: `flutter run` 后能在游戏场景中看到精灵/UI 贴图；能播放 BGM（可选）

### 任务 5: GameScene 页面与 GameProvider 双向绑定
- **说明**: `features/game/game_scene_page.dart` 的 `_gameWorld` 运行时 HP/MP/等级/经验自动同步到 `GameProvider`，UI PlayerStats widget 实时刷新
- **涉及文件**: `client/lib/features/game/game_scene_page.dart`、`client/lib/providers/game_provider.dart`
- **验收标准**: 游戏场景中 HP 变化后，顶部状态栏同步刷新

### 任务 6: AudioManager 真实化
- **说明**: 接入 flame_audio 播放 SFX（攻击/升级/死亡/拾取/传送门），BGM 按 mapId 映射 assets/audio 资源，音频缺失时静默而不是崩溃
- **涉及文件**: `client/lib/game/engine/audio_manager.dart`
- **验收标准**: 攻击时播放攻击音效，升级时播放升级音效
